# 理哩的个人空间 V2 前台页面原型设计文档

Version: 2.0
Date: 2026-06-12

# 一、页面原型目标
V2 前台从“按内容类型浏览”调整为“按博客结构浏览”。

核心变化：

+ 不再设置 `/travel`、`/essays`、`/journal`、`/moments`。
+ 改为首页文章流、分类、标签、归档、照片、关于。
+ 文章分类参考 `lilisong` 的个人化分类命名。
+ 前台增加注册、登录、退出入口，为未来评论、点赞功能预留账号体系。
+ 前台文章列表根据当前登录角色展示不同内容：管理员可见全部文章，访客和普通用户只看公开文章。

# 十四、注册 `/register`
## 页面目标
为普通用户创建账号，给未来评论、点赞等能力预留身份。

## 页面结构
```plain
注册

显示名，可选
密码
确认密码

已有账号？去登录
```

注册成功后：

+ 自动登录，或跳转登录页。
+ 默认角色为 `reader`。

# 十五、权限状态
V2 前台大部分页面公开访问，但文章数据需要根据角色过滤。

访问规则：

+ 未登录访客：只能看到公开且已发布文章。
+ 普通登录用户：只能看到公开且已发布文章。
+ 管理员登录用户：可以看到全部文章，包括仅自己可见、草稿、归档文章。

管理员前台标识：

+ `仅自己可见`
+ `草稿`
+ `已归档`

不显示：

+ 申请权限入口
+ 好友可见提示
+ 好友分组

后台入口只在管理员登录后显示。

# 十六、验收标准
+ 前台导航不再出现 Travel / Essays / Journal / Moments。
+ 首页以文章流为主。
+ 分类、标签、归档可以分别浏览文章。
+ 文章详情展示分类和标签。
+ Gallery 来自精选媒体。
+ 前台提供登录、注册、退出入口。
+ 访客和普通用户只能看到公开已发布文章。
+ 管理员登录后在前台可以看到全部文章，并能看到状态标识。
+ 整体观感接近私人博客，而不是分类内容门户。
